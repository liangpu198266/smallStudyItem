commit code step :<br>
1 touch README.md。这命令是添加一个文件。文件名叫：README.md;<br>
2 git  init ：这时初始化一个仓库。 成功后会有下面的提示：Initialized empty Git repository in d:.......<br>
3 git add . <br>
4 git commit -m ""<br>
5 git remote add origin git_url<br>
6 git push -u origin master 可能需要执行pull先<br>
